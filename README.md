:::: MENU 'Posts' ::::
I've demonstrated to fetch the data using a sample rest API from 'https://jsonplaceholder.typicode.com'. This is an open source API that can be used for practicing. Here I am using Blazor Server App using .NET 8 and Bootstrap for UI.
Basically the API includes data like 100 posts,	500 comments,	100 albums,	5000 photos,	200 todos,	10 users. Tried to fetch these data and display it on my web app.
1) When you run the app click on 'Posts' menu in Sidebar. It shows all the posts.
2) When you click on the 'Comments' of the post it shows all the comments related to that specific post.
3) Each post on the 'Posts' page has a name of the user and username below to it. When clicked on to the name of the user, it opens the User Profile page where you can find personal details of it and also some list of 'Albums' and 'To-do list' of that user.
